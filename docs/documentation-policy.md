# Documentation Policy

All project documents created from this point forward must be generated in two versions:

1. English working document under `docs/`.
2. Chinese reading copy under `C:\work\Vibe2\用于阅读`.

The Chinese reading folder is for the user only. Codex may write Chinese copies there, but should not read from that folder unless the user explicitly asks it to.

When an English document is updated, the corresponding Chinese reading copy should be updated in the same turn whenever practical.
